# 图表可视化能力

## 能力概述

系统支持将数据以图表形式可视化呈现，结合业务数据查询实现"查询 + 可视化"一体化分析。
共支持 **18 种**专业图表类型，涵盖趋势、对比、占比、分布、结构等多种场景。

---

## 支持的图表类型

### 趋势类

| 图表  | 适用场景                      |
|-----|---------------------------|
| 折线图 | 展示数据随时间的变化趋势，如订单量走势、销售额变化 |
| 面积图 | 趋势展示 + 总量对比，直观感受变化幅度      |

### 对比类

| 图表  | 适用场景                |
|-----|---------------------|
| 柱状图 | 分类数据的垂直对比，如各商品销量对比  |
| 条形图 | 分类数据的水平对比，名称较长时更清晰  |
| 雷达图 | 多维度综合评价对比，如商品多维属性比较 |

### 占比类

| 图表   | 适用场景              |
|------|-------------------|
| 饼图   | 各部分占整体的比例，如支付方式占比 |
| 矩形树图 | 带层级的占比数据展示        |
| 漏斗图  | 多阶段转化或流失分析        |

### 分布类

| 图表  | 适用场景               |
|-----|--------------------|
| 直方图 | 连续数据的频率分布，如价格区间分布  |
| 散点图 | 双变量关系分析，如价格与销量关系   |
| 词云图 | 词频/热度权重展示，如热销商品名词云 |

### 组合类

| 图表    | 适用场景                   |
|-------|------------------------|
| 双轴组合图 | 不同量纲数据同图对比，如订单量与金额同时展示 |

### 结构类

| 图表    | 适用场景          |
|-------|---------------|
| 思维导图  | 思维发散 / 知识结构展示 |
| 网络关系图 | 节点间关系连接展示     |
| 流程图   | 流程步骤与流向可视化    |

### 组织类

| 图表    | 适用场景          |
|-------|---------------|
| 组织架构图 | 组织层级结构展示      |
| 缩进树   | 目录 / 分类树形结构   |
| 鱼骨图   | 因果分析 / 问题根因追溯 |

---

## 图表选型指南

根据数据特征和展示目的选择合适的图表：

| 需求       | 推荐图表              |
|----------|-------------------|
| 展示时间趋势变化 | 折线图、面积图           |
| 分类数值对比   | 柱状图（类别少）、条形图（名称长） |
| 各部分占比分析  | 饼图、矩形树图           |
| 转化漏斗分析   | 漏斗图               |
| 数据频率分布   | 直方图               |
| 变量关系分析   | 散点图               |
| 词频/热度展示  | 词云图               |
| 多维评价对比   | 雷达图               |
| 双量纲同时展示  | 双轴组合图             |
| 知识/思维结构  | 思维导图              |
| 关系网络可视化  | 网络关系图             |
| 流程步骤展示   | 流程图               |
| 组织层级展示   | 组织架构图、缩进树         |
| 因果追溯分析   | 鱼骨图               |
