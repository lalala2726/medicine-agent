# 订单管理能力

## 能力概述

系统支持对药品商城订单进行全方位查询和管理，帮助运营人员快速定位目标订单、了解订单详情。

---

## 订单列表查询

按多种条件**搜索订单列表**，支持分页浏览。

### 支持的筛选维度

| 筛选维度     | 说明                                       |
| ------------ | ------------------------------------------ |
| 订单编号     | 精确匹配指定订单                           |
| 订单状态     | 待支付 / 已支付 / 已发货 / 已完成 / 已取消 |
| 支付方式     | 微信支付 / 支付宝                          |
| 配送方式     | 快递配送 / 到店自提                        |
| 收货人姓名   | 支持模糊搜索                               |
| 收货人手机号 | 精确匹配                                   |

### 使用示例

> "帮我查一下收货人叫张三的所有订单"
> "最新的10条已完成订单有哪些？"
> "用支付宝支付的待发货订单列表"
> "手机号 138xxxx1234 的订单"

---

## 订单详情查询

查看指定订单（可批量）的**完整详情信息**。

### 包含内容

- 订单基本信息（订单号、下单时间、状态等）
- 收货地址（姓名、电话、地址）
- 物流信息（物流状态、快递单号等）
- 商品明细（商品名称、数量、单价等）

### 使用示例

> "把这批订单号的详细信息都给我看看"
> "帮我看看订单 O20260101 的收货地址和物流状态"
> "这几个订单里面分别买了什么商品？"

---

## 常见业务场景

| 场景             | 推荐问法                       |
| ---------------- | ------------------------------ |
| 查找特定客户订单 | "帮我查收货人叫李明的订单"     |
| 跟踪发货进度     | "看一下已发货状态的订单有哪些" |
| 核对支付情况     | "今天微信支付的订单有几笔？"   |
| 查看订单详情     | "给我看一下这个订单的完整信息" |
| 批量查详情       | "帮我查这三个订单号的详细信息" |
