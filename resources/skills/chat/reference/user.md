# 用户管理能力

## 能力概述

系统支持管理端用户查询能力，覆盖用户列表、用户详情、用户钱包、钱包流水与消费信息，帮助运营和客服快速定位用户状态与资金记录。

---

## 用户列表查询

按多种条件分页查询用户列表。

### 支持的筛选维度

| 筛选维度   | 说明                         |
| ---------- | ---------------------------- |
| 用户 ID    | 精确匹配指定用户             |
| 用户名     | 支持模糊查询                 |
| 昵称       | 支持模糊查询                 |
| 头像 URL   | 精确匹配                     |
| 角色       | 按角色筛选（如 admin）       |
| 状态       | 按用户状态筛选（启用/禁用）  |
| 创建人     | 按创建人筛选                 |
| 分页参数   | page_num / page_size 分页查询 |

### 使用示例

> "查一下昵称里包含李的用户"
> "帮我看一下状态为启用的用户列表"
> "按角色 admin 查第一页 20 条用户"

---

## 用户详情查询

根据 `user_id` 查询单个用户详细资料。

### 使用示例

> "查看用户ID 1001 的详细信息"
> "帮我查一下这个用户的完整资料"

---

## 用户钱包与资金明细

### 用户钱包查询

根据 `user_id` 查询钱包余额和钱包状态。

### 钱包流水查询

根据 `user_id` 分页查询钱包流水记录（支持 `page_num/page_size`）。

### 消费信息查询

根据 `user_id` 分页查询消费记录（支持 `page_num/page_size`）。

### 使用示例

> "看一下用户1001的钱包余额"
> "查用户1001最近的钱包流水"
> "帮我查用户1001的消费记录"

---
