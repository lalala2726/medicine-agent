# 运营分析工具使用指南

## 工具概述

`analytics_tool_agent` 负责运营数据分析查询，内部包含 6 个工具，接收自然语言任务描述。

---

## 工具能力与返回数据

### 1. 运营总览

**用途**：获取商城核心运营指标的汇总数据。

**返回数据**：总订单数、总销售额、总用户数、退款笔数、退款金额等汇总指标。

### 2. 订单趋势

**用途**：按时间维度查看订单数量与金额的变化趋势。

**支持的周期**：按日（DAY）、按周（WEEK）、按月（MONTH）。未指定时默认按日。

**返回数据**：时间维度的趋势数据点列表，每个点包含时间标签、订单数量、订单金额。

### 3. 订单状态分布

**用途**：统计不同状态订单的数量与占比。

**返回数据**：各状态（待付款/已支付/已发货/已完成/已取消等）的订单数量和占比。

### 4. 支付方式分布

**用途**：统计不同支付渠道的使用情况。

**返回数据**：各支付方式的订单数量和占比。

### 5. 热销商品排行

**用途**：按销量统计最受欢迎的商品。

**支持参数**：TopN 数量（默认 10，范围 1-200）。

**返回数据**：热销商品排行列表，包含商品 ID、商品名称、销量、销售额等。

> **注意**：热销排行返回的**商品信息比较有限**（主要是 ID、名称、销量统计数据），不包含库存、价格、分类、药品说明书等信息 ——
> 需要时要拿商品 ID 去调 `product_tool_agent`。

### 6. 商品退货率

**用途**：统计商品的退货情况。

**支持参数**：TopN 数量。

**返回数据**：退货率排行列表，包含商品 ID、商品名称、退货数量、退货率等。

> **注意**：同热销排行，退货率数据不含完整商品信息，需要更多商品或药品信息时要调 `product_tool_agent`。

## 典型场景与调用方式

### 场景 1：运营数据总览

> 用户问法："最近运营数据怎么样？"、"商城整体情况看一下"、"总订单量和销售额是多少？"

调用：`analytics_tool_agent("获取运营总览数据")`

关注返回中的：总订单数、总销售额、总用户数、退款概况。

---

### 场景 2：查看订单趋势

> 用户问法："最近一周每天的订单量是多少？"、"按月看销售额变化"

调用：`analytics_tool_agent("查询按日的订单趋势")` 或 `analytics_tool_agent("查询按月的订单趋势")`

关注返回中的：趋势的变化方向（增长/下滑），峰值和谷值。

---

### 场景 3：订单状态分析

> 用户问法："现在各状态的订单分别有多少？"、"已取消的订单占比多少？"、"有多少未完成订单？"

调用：`analytics_tool_agent("获取订单状态分布")`

关注返回中的：各状态的数量和百分比。

---

### 场景 4：支付偏好分析

> 用户问法："大家主要用哪种方式付款？"、"微信和支付宝的使用比例"

调用：`analytics_tool_agent("获取支付方式分布")`

关注返回中的：各支付方式的使用量和占比。

---

### 场景 5：热销商品排行

> 用户问法："卖得最好的商品是什么？"、"热销排行前 10"、"最受欢迎的药品有哪些？"

调用：`analytics_tool_agent("查询热销商品排行 TOP10")`

关注返回中的：排名靠前的商品名称和销量。

---

### 场景 6：退货率分析

> 用户问法："退货率最高的商品有哪些？"、"哪些产品退货比较多？"

调用：`analytics_tool_agent("查询商品退货率排行")`

关注返回中的：退货率高的商品名称和退货率数值。

---

### 场景 7：自定义 TopN 排行

> 用户问法："给我看热销前 20 的商品"、"退货率最高的 5 个产品"

调用：`analytics_tool_agent("查询热销商品排行 TOP20")` 或 `analytics_tool_agent("查询退货率排行 TOP5")`

关注返回中的：指定数量的排行数据。

---

### 场景 8：运营复盘组合查询

> 用户问法："全面看看最近的运营情况，包括趋势和分布"

需要多步调用：

1. `analytics_tool_agent("获取运营总览数据")` → 概览指标
2. `analytics_tool_agent("查询按日的订单趋势")` → 趋势变化
3. `analytics_tool_agent("获取订单状态分布")` → 状态构成

最终汇总生成运营复盘报告。

## 跨域补查提示

运营分析数据经常需要**下钻到具体商品**了解更多信息：

| 用户后续提问                 | 提取字段        | 下一步调用                             |
|------------------------|-------------|-----------------------------------|
| "热销第一的商品库存还够吗？"        | 商品 ID       | product_tool_agent 查商品详情（关注库存）    |
| "退货率高的这两个药有什么问题？看看说明书" | 商品 ID       | product_tool_agent 查药品详情          |
| "热销商品的价格详情"            | 商品 ID       | product_tool_agent 查商品详情（关注价格/分类） |
| "退货率高的商品对应的售后记录"       | 商品名称/关联订单信息 | after_sale_tool_agent 查售后列表       |
| "做个热销排行的柱状图"           | 排行数据        | 使用 chart 技能直接生成图表                 |
| "订单趋势画成折线图"            | 趋势数据        | 使用 chart 技能直接生成图表                 |
